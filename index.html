Proyectos

La mayoría de proyectos en los que he participado forman parte en su mayoría del programa universitario donde me encuentro actualmente. Algunos destacados son:

Simulación de róvers en Marte

Este proyecto consiste en un simulador de dos róvers en Marte, uno solar y otro eólico. Hay cráteres en Marte donde se encuentran materias que serán sensados. El usuario mediante comandos, se puede mover alrededor del planeta, puede planificar rutas, entre otros. Además, se pueden usar los dos róvers al mismo tiempo gracias al Multithreading.

Imágenes

Aspectos Técnicos

  • Lenguaje usado: Java
  • Uso de: interfaces, hilos (multithreading), JavaFX, Archivos serializados

Link del repositorio: Proyecto-POO-G2

Catálogo de Videojuegos

El proyecto consiste en simular un catálogo de videojuegos estilo Steam, donde se hace uso de Estrucutra de Datos como las listas circulares enlazadas. Además, hay dos vistas: una para el usuario y otra para el administrador. Por último, existen diversas formas de búsqueda de videojuegos.

Aspectos Técnicos

  • Lenguaje usado: Java
  • Uso de: EDD (listas enlazadas), JavaFX, Archivos serializados

Link del repositorio: Proyecto-P1-Estructuras-G6

Visualizador de Archivos

Consiste en visualizar los archivos de una manera más general, donde el tamaña de un archivo/directorio ocupará más o menos espacio en el visualizador. El comportamiento del programa está basado en árboles multicaminos. Tiene dos modos de visualización: una por archivos y otra por extensiones de archivo.
